
## 安装步骤
### 使用laravel7-layuiadmin
- git clone  https://github.com/kinchuam/laravel7-layuiadmin.git
- 复制.env.example为.env
- 配置.env里的数据库连接信息
- composer update
- php artisan migrate
- php artisan db:seed
- php artisan key:generate
